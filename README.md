# vue-sneakers

Современный сайт-магазин с поиском, фильтрацией нужных товаров.  
Реалиизовано добавление товара в избранное, в корзину, и отправка заказа на сервер

## Технологии

- Vue 3
- Vue Router
- TailwindCSS
- Axios
- @formkit/auto-animate
- Composition API / Options API
- ESLint
- Prettier

### Доработать

- профиль
- мобильная верстка
